= Drive
:description: The Drive allows you to store and carry a filesystem.

[cols="1,3"]
|===
|Itemtype
|Item
|===

[cols="a"]
|===
|Inventory

|image::items/Drive_T1.png[300,300,role="left"]
|===


The Drive can be placed into a drive slot in a Computer, which allows it to read and write data to the Drive.
The Drive has a GUID to identify the filesystem.
Data on the Drive is persistent, so you can use a single drive on different Computers.
