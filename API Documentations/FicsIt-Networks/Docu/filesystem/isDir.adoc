=== `bool isDir(string path)`

Checks if given path refers to a directory.

Parameters::
+
[cols="1,1,4a"]
|===
|Name |Type |Description

|path
|string
|path you want to check if it refers to a directory
|===

Return Values::
+
[cols="1,1,4a"]
|===
|Name |Type |Description

|bool
|bool
|returns true if path refers to a directory
|===
