=== `bool.. isNode(string...)`

For each given string, returns a bool to tell if string is a valid node (file/folder) name.


Parameters::
+
[cols="1,1,4a"]
|===
|Name |Type |Description

|string...
|string...
|node-names you want to check.
|===

Return Values::
+
[cols="1,1,4a"]
|===
|Name |Type |Description

|bool...
|bool...
|True if the corresponding string is a valid node-name.
|===
